Contents:

Folder potsherds contains a set of 207 3D surfaces in OFF format. They correspond to external scanned views of potsherds from the Teoitihuacan culture, as presented in:

Edgar Roman-Rangel, Diego Jimenez-Badillo, and Stephane Marchand-Maillet.
"Classification of Archaeological Potsherds using Histograms of Spherical Orientations".
ACM Journal on Computing and Cultural Heritage (JOCCH). 9(3):Online first. September, 2016.

The accompaning files dataInfo.mat and dataInfo.xlsx have the list of ID classes and descriptions for the set of potsherds, in matlab and excel format respectively.

Permision to use this models with research purposes is granted.
